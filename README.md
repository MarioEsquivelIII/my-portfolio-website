# My Portfolio

Personal portfolio website built with **React**, **Vite**, and **Bootstrap**. Deployed automatically to **GitHub Pages** via GitHub Actions on push to `master`.

**Live site:** [marioesquiveliii.github.io/my-portfolio-website](https://MarioEsquivelIII.github.io/my-portfolio-website)

## Tech Stack
- React 19 + React Router
- Vite
- Bootstrap / React Bootstrap
- EmailJS (contact form)

## Development

```bash
npm install
npm run dev
```
